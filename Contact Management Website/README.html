# Contact Management Website ## Project languages the Contact Management Website was developed developed using HTML, CSS, and JavaScript. ## Project Description This website is designed for storing and managing personal contacts through a simple and user-friendly interface. Users can log in using their username and password, add, update, and search contacts efficiently. ## Features - **Login System**: Users log into the site using their username and password. - **View Contacts**: Users see a list of contacts they have added. - **Add Contact**: Option to add a new contact with details such as name, phone number, and email. - **Update Contact**: Ability to update contact details like phone number or email. - **Search Specific Contact**: The ability to search for a specific contact by name or other details. ## System Directories - **css**: Directory containing CSS files for site styling. - **html**: Directory containing the main HTML file of the site. - **javascript**: Directory with JavaScript files to support site functionality. - **images**: Directory containing images used on the site. ## Running the Site 1. Download all project directories. 2. Navigate to the `html` directory. 3. Open the `spa-demo.html` file. 4. Enter the login or registration page, fill in the details, and access the site. ## Author Mally Grinvald Good luck!
